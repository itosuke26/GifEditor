# GifEditor 
 
GifEditorは、GIFアニメーションを編集するためのツールです。 
 
## 機能 
 
* GIFアニメーションのフレームレート調整 
* GIFアニメーションのカラーパレット最適化 
* GIFアニメーションの切り抜き 
* GIFアニメーションのリサイズ 
* GIFアニメーションのファイルサイズ削減 
 
## インストール 
 
1. [こちら]([ClickOnceで発行されたアプリケーションのURL])からインストーラーをダウンロードします。 
2. インストーラーを実行し、アプリケーションをインストールします。 
 
## 使用方法 
 
1. アプリケーションを起動します。 
2. GIFアニメーションファイルを読み込みます。 
3. 必要な編集操作を行います。 
4. 編集結果を保存します。
